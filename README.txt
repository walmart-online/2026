Walmart Catalog App — Landing Page
==================================

CONTENTS:
- index.html — single-page site (Walmart style, EN)
- Walmart.exe — placeholder file for download (replace with your real installer)

BEFORE PUBLISHING / FACEBOOK ADS:
1. Replace "https://yoursite.com/" in index.html with your real site URL (og:url).
2. Add image og-image.jpg (1200×630 px) for Facebook preview; put it in the same folder and set og:image to your URL + "/og-image.jpg".
3. Replace the placeholder Walmart.exe with your actual Windows installer so users get the real app when they click Download.

The page is ready for Windows PC users and includes Open Graph meta tags for Facebook ads.
